This is simple Blogging Platform where users can post their blogs and view blogs of other users.

All the data is stored in mongodb so you will need to connect your local mongodb with this project. Simply add url from your mongodb compass into server.js file in server directory.

To run this project follow below steps.

Terminal 1:
1. cd server
2. npm install
3. nodemon server.js OR node server.js

Terminal 2:
4. cd client
5. npm install
6. npm start

That's it you will be directed to login page. If this is your first time and you are not registered then simply click on "Register Now" button
